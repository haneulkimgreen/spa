css blend(background-blend-mode, mix-blend-mode)

css에서 이미지 포토샵 효과, 이미지 위에 컨텐트에 효과를 줄 수 있음.

생활코딩 설명 https://www.opentutorials.org/course/2418/13667

mix-blend-mode

scroll_back_effect에서 활용

scroll_back_effect를 이용해 fixed로 고정된 이미지 배경 위를 컨텐트가 지나갈때 효과를 주었음.

검은 배경 위를 지나갈때 글자 색이 흰색으로 바뀜.

배경에 css mix-blend-mode: difference;
